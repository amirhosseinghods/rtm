# RTM روی هاستِ بدونِ پایتون (GitHub Actions + PHP)

هاستت پایتون ندارد؟ مشکلی نیست. **محاسبه روی GitHub Actions** (رایگان) انجام می‌شود و
هاستِ cPanel فقط سایت را نشان می‌دهد و هر چند دقیقه دیتا را `git pull` می‌کند.

```
GitHub Actions (هر ۱۵ دقیقه، پایتونِ رایگان)            هاستِ cPanel تو (فقط PHP/استاتیک)
  • دیتای زنده می‌گیرد                                     • git pull هر ۱۵ دقیقه (دیتای نو)
  • signal/zone/assistant/RSI/پیش‌بینی را می‌سازد   ──►   • index.html + app.js دیتا را نشان می‌دهد
  • recorder/optimize را اجرا می‌کند (یادگیری)            • کندل + قیمتِ زنده مستقیم از Binance (JS)
  • JSON را در site/data/ کامیت می‌کند                    • ورود با .htpasswd ، ژورنال در MySQL
```

**هیچ پایتونی روی هاست لازم نیست. توکن هم لازم نیست** (ریپو عمومی است، pull بدونِ کلید).

## راه‌اندازی
1. **ریپوی عمومی** بساز و این پروژه را در آن push کن (یک‌بار، از کامپیوترِ خودت).
2. در گیت‌هاب → **Settings → Actions → General** اطمینان حاصل کن Actions فعال است (workflow
   در `.github/workflows/compute.yml` خودکار هر ۱۵ دقیقه اجرا می‌شود؛ یک‌بار هم با تبِ Actions →
   **Run workflow** دستی بزن تا `site/data/` پر شود).
3. در cPanel یک **زیردامنه** بساز که document root آن پوشهٔ `site/` باشد.
4. با cPanel → **Git Version Control** ریپوی عمومی را در همان پوشه کلون کن (یا File Manager).
5. در cPanel → **MySQL Databases** یک دیتابیس بساز.
6. در مرورگر برو به `https://your-subdomain/installer.php`، فرم را پر کن:
   - دیتابیس (هاست/نام/یوزر/پسورد)
   - **یوزر/پسوردِ سایت** (ورود)
   - آدرسِ ریپوی عمومی + هر چند دقیقه pull شود
7. «نصب کن» → DB، ورود (.htpasswd)، و **کرانِ `git pull`** را می‌سازد. بعد **`installer.php` را حذف کن**.

تمام. سایت با ورود محافظت می‌شود، هر ۱۵ دقیقه از گیت‌هاب آپدیت می‌شود، و خودبهینه است
(optimize روی Actions). داشبورد، نواحی، دستیار، RSI و پیش‌بینی از JSON می‌آیند؛ کندل و قیمتِ
لحظه‌ای مستقیم از Binance در مرورگر.

## نکات
- **طلا:** نمادِ «طلا (PAXG · لحظه‌ای)» کندل‌هایش از Binance می‌آید (لحظه‌ای). XAUUSD (تأخیری)
  کندل‌هایش را Actions در `data/ohlcv_XAUUSD_*.json` می‌گذارد.
- **تاریخچهٔ چارت:** در این حالت کندل از Binance تا ۱۰۰۰ بار می‌آید (روی H1 ≈ ۴۱ روز، روی M15 ≈ ۱۰
  روز). برای دیدِ ماه‌ها به H1/H4 سوئیچ کن.
- **حجمِ ریپو:** هر اجرا JSONها را کامیت می‌کند؛ هر چند ماه یک‌بار می‌توانی history را squash کنی.
- **امنیت:** فقط DB-pass و پسوردِ سایت روی هاست است (توکن لازم نیست). `installer.php` را بعد از نصب حذف کن.
- اگر مرورگرت به Binance دسترسی نداشت (فیلتر)، کندل/قیمت نمی‌آید ولی تحلیل/نواحی/RSI از JSON کار می‌کنند.
